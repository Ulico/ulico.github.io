# adrianrusso.com

Personal website of Adrian Russo — software engineer and jazz musician.

A single-page portfolio built with React, TypeScript, and Vite: an About section,
performance videos (click-to-play YouTube/Facebook embeds), and a grid of projects
pulled from real work on GitHub.

## Development

```bash
npm install
npm run dev      # dev server at http://localhost:5173
npm run build    # type-check + production build to dist/
npm run preview  # serve the production build locally
```

Deployed to GitHub Pages at [adrianrusso.com](https://adrianrusso.com) via the
workflow in `.github/workflows/`.
